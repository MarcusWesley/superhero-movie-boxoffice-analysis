Superhero Movie Financial Analysis

Interactive data visualizations exploring various financials for the live action Superhero Genre (2025 USD)

Project Overview

This project analyzes superhero movie financials across MCU, DC, FOX, and SONY franchises, using inflation-adjusted data (base year: 2025). Built with Python, Pandas, and Plotly, it features interactive charts to explore revenue trends, profitability, ROI, and budget breakdowns. Click below to view the visualizations.
